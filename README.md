# Maher Media Portfolio Site

A polished static portfolio website for Blake Maher / Maher Media.

## How to use
1. Extract the ZIP.
2. Upload the contents of this folder to the `Maher-Media` GitHub repository.
3. Deploy through Netlify, Vercel, or GitHub Pages.

## Files
- `index.html` — full website structure and text.
- `styles.css` — responsive design, animations, layout, and visual system.
- `script.js` — reveal animations, counters, cursor effects, tilt cards, and email-estimate form.
- `assets/` — resume preview, headshot, Delta Gamma Iota screenshots, and campaign graphics.
